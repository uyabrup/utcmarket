Файл ru_FULL.po является полным переводом сборки. Изначально сдобка переводилась только с помощью этого файла, однако потом переводы были разделены. При разделении возможна потеря некоторых строк, вы можете загрузить после установки полный файл для перевода их.В следующих релизах эта проблема будет решена.
